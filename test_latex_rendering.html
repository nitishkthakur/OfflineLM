Mixed Content Examples

Code and Math together:

function quadratic(a, b, c) {
    let discriminant = b*b - 4*a*c;
    return (-b + Math.sqrt(discriminant)) / (2*a);
}

This implements the quadratic formula: $x = \frac{-b + \sqrt{b^2 - 4ac}}{2a}$
